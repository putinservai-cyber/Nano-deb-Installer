## Features

*   **GUI-based Installation:** Provides a user-friendly graphical interface for installing Debian packages.
*   **Self-Update Capability:** Includes functionality to check for and apply updates to the installer itself.
*   **Installation Wizards:** Guides users through the setup process using structured wizards.
*   **Report Generation:** Capable of generating reports related to the installation process.
*   **Donation Page Integration:** Includes a dedicated page for supporting the project.
*   **Security Focus:** Utilizes a dedicated security module for handling sensitive operations.
